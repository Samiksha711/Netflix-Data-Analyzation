# 📺 Netflix Data Analysis using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The objective is to uncover meaningful insights into Netflix's content library through data cleaning, visualization, and statistical analysis.

---

## 📂 Dataset

- Dataset: Netflix Movies and TV Shows
- Source: Kaggle
- Records: 8,800+ titles
- Features: Title, Type, Director, Cast, Country, Rating, Duration, Genre, Date Added, Release Year, etc.

---

## 🎯 Objectives

- Analyze Movies vs TV Shows
- Identify top contributing countries
- Explore rating distribution
- Analyze genre popularity
- Study movie duration trends
- Examine yearly Netflix growth
- Identify top actors and directors
- Generate business insights from data

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed

- Data Cleaning
- Missing Value Handling
- Feature Engineering
- Exploratory Data Analysis
- Correlation Analysis
- Data Visualization

---

## 📈 Key Insights

- Movies dominate Netflix's content library.
- TV-MA and TV-14 are the most common ratings.
- The United States contributes the highest number of titles.
- Drama and International Movies are the most popular genres.
- Netflix experienced rapid content growth after 2015.
- Movie durations mostly range between 80–120 minutes.
